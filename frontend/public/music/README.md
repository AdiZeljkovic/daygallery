# Pozadinska muzika za pozivnice

Ovdje idu preset numere koje šef bira u adminu (Pozivnice → Muzika).
Player na javnoj pozivnici traži tačno ova imena fajlova. Ako fajl ne postoji,
muzika se jednostavno ne pušta (bez greške).

Očekivani fajlovi (dodaj **royalty-free** mp3, ~2–4 MB, instrumentalne):

- `royal-1.mp3`  → izbor "Royal 1"
- `royal-2.mp3`  → izbor "Royal 2"
- `royal-3.mp3`  → izbor "Royal 3"

Napomena o licenci: koristi isključivo numere za koje imaš prava (npr. YouTube
Audio Library, Pixabay Music, Uppbeat). Ne ubacuj komercijalne pjesme bez licence.

Muzika kreće tek kad gost klikne "Otvori pozivnicu" (zbog autoplay pravila
preglednika) i može se utišati plutajućim dugmetom (nota) dolje-desno.
